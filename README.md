> **요약**
> 
- 커핑은 근처 지역에서 좋아하는 원두를 찾아 즐길 수 있는 원두 기반 카페 검색 서비스입니다.
- `일반 사용자`는 지역 기반 검색을 통해 원두 기반으로 카페를 찾고 검색하실 수 있습니다.
- `카페 관리자`는 본인 가게에서 보유하고 있는 원두를 등록하고, 가게 정보도 등록하실 수 있습니다.

> **역할**
- 인원

    - FE 3명
    - DE 1명
- 개발환경
    - FE
        - React( +Typescript )

        - Axios
        - Recoil
        - React-query
        - TailwindCSS

        - SpringBoot
